## Propuesta


El proyecto consiste en el desarrollo de un videojuego plataformero 2D en Unity, compuesto por 2 niveles definidos. La propuesta se centra en una mecánica principal: el uso de un yoyo que el jugador puede lanzar y recuperar, el cual cumple múltiples funciones específicas dentro del diseño del juego, incluyendo infligir daño a enemigos, agarrar y lanzar objetos del entorno y permitir un sistema de desplazamiento basado en balanceo entre diferentes puntos, donde el  yoyo se pega al punto y el jugador empieza a balancearse, la idea es que este el balanceo principal, pero que en ocasiones hayan puntos donde se un impulso tipo catapulta para llegar a otras secciones del mapa. Estas mecánicas constituyen el núcleo del gameplay y se diferencian de las mecánicas base del género, como el movimiento lateral, el salto y la interacción con plataformas. El juego aborda temas de acción y navegación en entornos con obstáculos, donde la habilidad del jugador para usar el yoyo de forma estratégica define su progreso. Cada nivel incluirá un conjunto definido de obstáculos como plataformas móviles, zonas de caída, secciones que requieren uso del yoyo para avanzar y enemigos como obstáculos dinámicos, también tendrán consumibles para recuperar vida o coleccionables para los puntos. El sistema de estados del jugador incluye movimiento, saltó, uso del yoyo (activo/inactivo/enganchado) y un sistema de salud representado mediante una barra de vida visible, la cual disminuye al recibir daño y provoca el reinicio del nivel al agotarse. En cuanto a los enemigos, se implementarán 3 tipos claramente diferenciados: un enemigo saltador que sigue una ruta fija mediante saltos, un enemigo terrestre que sigue un recorrido pero persigue al jugador al detectarlo, y un enemigo volador que sigue al jugador y ataca a distancia con proyectiles. De esta manera,  la idea es jugar con los parámetros que tendría cada enemigo, como potencia de salto, velocidad, que el volador lance diferentes cantidades de proyectiles o algún efecto negativo como invertir controles, lanzarlo atrás fuerte si el jugador toca es recibe daño por un enemigo.
